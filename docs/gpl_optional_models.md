# Optional GPL Models

## Policy
- GPL-licensed models are optional installs only.
- They are not bundled with default distribution artifacts.
- Install requires explicit user acceptance in Model Manager.

## Current GPL Entries
- `Swin2-MoSE` (GPL-2.0)
- `DSen2` (GPL-3.0)

## Notes
- Review source repository license terms before installing.
- If license terms are not acceptable, do not install these models.
